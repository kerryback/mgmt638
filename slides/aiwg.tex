\begin{frame}{Claude Code (Introduced for Windows July 2025)}
\begin{itemize}
\item Uses Claude Opus or Claude Sonnet as LLM
\item Works beside you on your computer, providing information and assistance
\begin{itemize}
\item Can read, edit, and create files on your computer (no manual uploading and downloading)
\item Can search the web and download files
\item Can run terminal commands, including running python scripts and compiling LaTeX docs
\item Can use Python to create Excel, Word, and PowerPoint docs
\item Can create and edit Jupyter notebooks with code, markdown, and visualizations
\end{itemize}
\end{itemize}
\end{frame}

\begin{frame}{MGMT 638 Slide 12/01/2025}
Ask Claude:
\vspace{0.5cm}

    \begin{enumerate}
    \item The risk-free rate is 2\%.  My portfolio has an expected return of 10\% and a standard deviation of 15\%.  I'm considering moving some money to a new asset that has a standard deviation of 30\%, a correlation with my portfolio of 60\%, and an alpha relative to my portfolio of 5\%.  How can I combine the new asset and the risk-free asset with my current portfolio to get an expected return higher than 10\% while maintaining the standard deviation at 15\%?
    \item Put this analysis in a Jupyter notebook, doing the calculations in Python.
    \end{enumerate}

\item Read portfolio\_returns.csv.  What are the mean returns of the portfolios?
\item Plot the cumulative returns of the portfolios.
\item Plot the drawdowns of the portfolios.  What is the maximum drawdown of each?
